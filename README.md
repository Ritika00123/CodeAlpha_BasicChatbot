# Basic Chatbot

## Description
A simple rule-based chatbot built using Python. It responds to predefined user inputs such as greetings and common questions.

## Features
- Greeting responses
- Conversation handling
- Exit command
- Simple rule-based logic

## Technologies Used
- Python 3

## How to Run
1. Install Python 3.
2. Open terminal or command prompt.
3. Run:

python BasicChatbot.py

## Supported Commands
- hello
- how are you
- what is your name
- who made you
- bye

## Example

You: hello
Bot: Hi!

You: how are you
Bot: I'm fine, thanks!
